# Biology Rules

The project is a conceptual educational simulation, not a wet-lab protocol.
Keep biology student-safe, simplified, and function-first.

## Global Rules

- Every component must support a biological function or teaching interaction.
- Do not imply impossible processes for the organelle.
- Keep organelle relationships clear: nucleus to ribosomes, rough ER to Golgi,
  mitochondria to ATP demand, membrane to transport, lysosomes to recycling.
- Student-facing explanations should be concise and accurate.

## Core Function Map

| Organelle | Required Concept |
| --- | --- |
| Nucleus | DNA storage, transcription, pores, control signals |
| Mitochondrion | ATP production, inner folds, oxygen/glucose dependency, stress |
| Golgi | Cargo modification, sorting, vesicle dispatch |
| Rough ER | Ribosome-studded protein folding and Golgi handoff |
| Smooth ER | Lipid synthesis and calcium/stress support |
| Ribosome | mRNA translation into proteins |
| Lysosome | Digestion, recycling, waste handling |
| Peroxisome | Oxidative stress reduction and lipid handling |
| Cytoskeleton | Structure, transport tracks, spindle support |
| Centrioles | Microtubule organization and mitotic coordination |
