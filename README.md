# 3D Zombie Attack Game

A first-person shooter zombie survival game built with Three.js featuring realistic textures and atmospheric night gameplay.

## Features

- **First-person shooter mechanics** with WASD movement and mouse look
- **Realistic zombies and zombie dogs** with hiding/ambush behavior
- **Two weapons**: Rifle (10 bullets) and axe (melee)
- **Headshot mechanics** - 1-hit kill with head explosion effect
- **Night atmosphere** with fires, lamps, torches, and player flashlight
- **Large 2-story building** with functional stairs to explore
- **Detailed procedural textures** for all game elements
- **Wave-based spawning system** with progressive difficulty
- **Death/respawn system** - 5 deaths = game over

## How to Play

1. Open `zombie_attack_3d.html` in a web browser
2. Click to lock the mouse pointer
3. Use **WASD** to move
4. Use **mouse** to look around
5. **Left click** to shoot/swing axe
6. Press **1** for rifle, **2** for axe
7. Press **R** to reload or restart
