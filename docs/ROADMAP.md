# Roadmap

## Phase 1: Repository Foundation

- Keep source, docs, backend, bridge scripts, and organelle briefs in one clean
  GitHub repository.
- Exclude local caches, virtual environments, databases, and binary Roblox
  exports from version control.
- Maintain repeatable build and validation commands.

## Phase 2: Organelle Assembly

- Promote organelle project briefs into production Luau modules one structure at
  a time.
- Start with membrane, cytoplasm, cytoskeleton, and nucleus because they define
  scale, navigation, and cell identity.
- Preserve each organelle as a self-contained movable model.

## Phase 3: Learning Interactions

- Add hotspot flows, guided tours, quizzes, and process animations.
- Link Roblox interactions to backend simulation state where useful.
- Keep explanations suitable for A-level and first-year university learners.

## Phase 4: Agentic Simulation

- Use deterministic backend agents for fast-loop biology state.
- Add optional LLM director behavior only through validated, structured
  directives.
- Keep Roblox as renderer and input collector, not the canonical state owner.
