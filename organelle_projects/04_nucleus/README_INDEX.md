# Nucleus Reverse Component Pack v3

This pack expands the nucleus organelle into a much deeper reverse-engineered Blender production structure.

## What “reverse” means here

Start with the final cohesive organelle:
`whole animal cell context → nucleus as one assembled organelle`

Then reverse-break it into:
`functional groups → individual Blender mesh pieces → prompts → integration rules → review checkpoints`

## Folder guide

- `per_component_prompts/` — one Blender AI prompt file per component.
- `per_group_prompts/` — group-level prompts for building systems like envelope, NPCs, lamina, genome, transport.
- `manifests/` — AI-readable JSON and CSV component data.
- `review/` — controls, checkpoints, and pass criteria.
- `integration/` — Roblox assembly, build order, hierarchy, and visual reference prompt.
- `copy_paste_prompts/` — prompts intended to paste directly into a Blender-code AI.

## Production target

Animal eukaryotic nucleus, stylized-realistic, colorful, biologically accurate, Roblox-friendly, animation-ready.
