# Organelle Project Template

Copy this folder when starting a new organelle project.

## Definition Of Done

- The organelle builds as one movable Roblox `Model`.
- It has a stable pivot/root and can be repositioned with `Model:PivotTo()`.
- Major biological structures are named as children of the model.
- The project includes a spec file with scale, colors, educational hotspots, and
  performance notes.
- The module returns hotspot descriptors using the shared build contract.
- It can be assembled through `OrganelleRegistry.lua` without editing unrelated
  organelle code.
