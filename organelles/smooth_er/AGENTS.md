# Smooth ER Agent Contract

Build Smooth ER as an organic, tubular membrane network for a stylized-realistic
animal cell. The model must read as smooth endoplasmic reticulum, not rough ER:
no ribosome coating and no flat sheet dominance.

Required biological ideas:
- lipid synthesis along smooth tubular membrane
- detox reaction sites on curved membrane surfaces
- calcium storage pockets inside expanded tube regions
- transition handoff cues toward rough ER/Golgi trafficking context
- inspectable hotspots for teaching and animation

Reject plain cylinders, toruses, spheres, or decorative beads. Visible
biological meshes must be generated organic forms with asymmetry, rounded
geometry, procedural texture, thickness cues, and function metadata.
